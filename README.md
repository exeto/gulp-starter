# gbp [![Dependency Status][depstat-image]][depstat-url]

Gulp boilerplate.

## Install

```bash
$ git clone https://github.com/exeto/gbp.git
$ cd gbp && npm i
```

## Usage

```bash
# Development
$ npm start

# Production
$ npm run build
```
